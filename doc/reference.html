Public Functions Documentations

Examples

uFlex - Use to resume session, login user and handles auto login
register - Use to register new user
update - Use to update any field on the database
addValidation - Use to valiate fields when using register or update
activate - Use to activate unactive accounts
pass_reset - Use to generate a hash to reset a forgotten password
new_pass - Use to set a new password with the hash generated from pass_reset
logout - Use to logout user and destroy any autologin cookie

For Multiple Entries:
$name = (array) takes an object in following format:
array(
"username" => array(
    "limit" => "3-15",
