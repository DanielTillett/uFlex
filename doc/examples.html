Some Examples to get Started with the uFlex Class

Starting the uFlex object

The uFlex object needs to be started on every page you want to authenticate 

    
        //This start the object and Resume the users current session
        $user = new uFlex();

Sample Login form

PHP


    

    
    
    

    
        

            This would be included in your_login_process_page.php file which will proccess the login request.
        

        
            $username = $_POST['username'];
            $password = $_POST['password'];
            $auto = $_POST['auto'];  //To remember user with a cookie for autologin
            
            $user = new uFlex($username,$password,$auto);
            
            //not required, just an example usage of the built-in error reporting system
            if($user->signed){
                echo "User Successfully Logged in";
            }else{
                //Display Errors
                foreach($user->error() as $err){
                        echo "<b>Error:</b> {$err} <br/ >";
                }
            }

            This would be included in "your_registration_process_page.php" file which will proccess the registration request.
            Because the $_POST array we are going to get will match the object for the register method we will pass the whole $_POST
            array to the register method.
        

        
        
            $user = new uFlex();
            
            $user->register($_POST,false);
            
            //OR fancier
            
            if($user->register($_POST,false)){
                echo "User Registered";
            }else{
                //Display Errors
                foreach($user->error() as $err){
                        echo "<b>Error:</b> {$err} <br/ >";
                }
